Question-1: What are the different ways to select an element in the DOM ?

There are many ways to select an element in the DOM. We can use getElementsByTagName (To select elements by tag name), getElementsByClassName (To select elements by class name), getElementById (To select a single element by an id), querySelector (To select the first child of an element) and querySelectorAll (To select the elements by using both the class and id).

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The innerHTML contains all the HTML structure of an element. The innerText contains only the plain text of an element but textContent contains text along with any hidden text.

Question-3: What is event delegation in the DOM ?

Event delegation is used to handle many child elements by targeting only one element at a time.

Question-4: What is event bubbling in the DOM ?

Event bubbling is a process where events travels into elements by propagating in a reverse way to their parent elements.

Question-5: How do you create, add, and remove elements using JavaScript ?

We can create elements by using document.createElement() property, add elements by using document.append() property and remove elements by using document.remove() property.
